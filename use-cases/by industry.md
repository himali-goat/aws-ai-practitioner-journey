# AWS AI Services to Scenaerio
### Amazon Rekognition
Image and video analysis — object detection, facial recognition, content moderation, celebrity detection, PPE detection.
Use when: scenario involves analyzing images or video
### Amazon Comprehend
Natural language processing — sentiment analysis, entity extraction, key phrase detection, language detection, topic modeling.
Use when: scenario involves understanding text or documents
### Amazon Transcribe
Automatic speech-to-text — converts audio/video files or live streams into text. Supports custom vocabulary.
Use when: scenario involves converting speech or audio to text
### Amazon Polly
Text-to-speech — converts text into lifelike speech. Supports multiple languages and voices.
Use when: scenario involves converting text into audio/voice
### Amazon Textract
Document text and data extraction — reads text, tables, and forms from scanned documents and PDFs beyond simple OCR.
Use when: scenario involves extracting data from forms, invoices, or PDFs
### Amazon Translate
Neural machine translation — translates text between languages in real time or batch mode.
Use when: scenario involves translating content between languages
### Amazon Forecast
Time-series forecasting — predicts future values like demand, inventory needs, or financial metrics using ML.
Use when: scenario involves predicting future quantities over time
### Amazon Personalize
Real-time personalization and recommendations — the same technology used on Amazon.com for product recommendations.
Use when: scenario involves personalizing content or product recommendations
### Amazon Kendra
Intelligent enterprise search — uses ML to search across internal documents and return accurate, natural language answers.
Use when: scenario involves searching internal knowledge bases or documents
